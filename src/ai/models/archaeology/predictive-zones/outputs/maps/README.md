## ‚öôÔ∏è Map Generation Workflow

```mermaid
flowchart TD
  A["Model Predictions (Feature Stack)"] --> B["Raster Creation (GeoTIFF ¬∑ EPSG:4326)"]
  B --> C["Zonal Thresholding & Classification"]
  C --> D["Vectorization (GeoJSON Polygons)"]
  D --> E["Masking (Restricted Heritage Zones)"]
  E --> F["Governance Validation + Telemetry Merge"]
```

### Steps:
1. **Raster Creation:** Generates continuous probability raster (0‚Äì1).  
2. **Classification:** Categorizes raster cells into predictive classes.  
3. **Vectorization:** Converts significant zones into polygons.  
4. **Masking:** Applies ethical redaction for Indigenous or heritage sites.  
5. **Governance Validation:** FAIR+CARE Council approves before release.
