## üìò Overview

The `logs/` directory records **every ingestion, entity linking, and provenance sync event** within the KFM Graph subsystem.  
Each file acts as an immutable ‚Äúlab notebook‚Äù entry for the data pipeline, aligning with the **Master Coder Protocol (MCP)** principle of verifiable, documented computation.

Logs and audit files provide:
- Full provenance of graph insertions and updates.  
- Timestamped transaction chains.  
- Cryptographic checksums (SHA-256) for verification.  
- Summary of successful blockchain/IPFS sync events.  
- AI-linked entity resolution statistics (confidence scores, anomalies).

| **focus_link_metrics.json** | Entity linking confidence scores (NER ‚Üí Graph linking model). | JSON | `link_entities.py` + Focus Transformer v2 |
