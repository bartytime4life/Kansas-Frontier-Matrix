## ğŸ“˜ Overview

The **Data Docs** are automatically generated HTML validation reports produced by Great Expectations during checkpoint execution (`stac_items.yml`).  
They provide:

- A visual summary of **validation results**
- Schema & rule failures
- CARE/governance violations
- Lineage & provenance references
- Links to STAC Item IDs that failed validation
- Run metadata & Expectation Suite versions

These pages are stored temporarily in CI and optionally published as GitHub workflow artifacts.
