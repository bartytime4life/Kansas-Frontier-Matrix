## ğŸ§  Map Types & Color Space Rules

| Map | Purpose | Color Space | Typical Format | Notes |
|---|---|---:|---|---|
| `baseColor` | Diffuse/albedo color | **sRGB** | `.ktx2` / `.jpg` | No baked lighting if possible (avoid heavy AO shadows baked into albedo). |
| `normal` | Surface detail lighting | **Linear** | `.ktx2` / `.png` | Tangent-space normals; ensure correct Y orientation for your renderer pipeline. |

| `height` *(opt)* | Parallax/displacement | **Linear** | `.png` | Use sparingly on web (costly). |
| `emissive` *(opt)* | Glow/self-lit | **sRGB** | `.ktx2` / `.png` | Only if the asset truly emits light. |
| `opacity` *(opt)* | Cutout/transparency | **Linear** | `.png` | Prefer alpha in baseColor when possible. |
