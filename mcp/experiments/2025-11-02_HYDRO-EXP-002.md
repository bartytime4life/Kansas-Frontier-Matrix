# üíß **HYDRO-EXP-002 ‚Äî Hydrology Multi-Source Fusion Validation**  
`mcp/experiments/2025-11-02_HYDRO-EXP-002.md`

**Purpose:**  
Validate the multi-source fusion methodology that integrates **USGS**, **USACE**, **Mesonet**, and **ERA5**‚Äìderived hydrology predictions into a unified flow/inflow/outflow series for Kansas basins.

</div>

# üìò 1. Objective / Hypothesis

### Objective  
Evaluate the fusion algorithm used in Hydrology Seq2Seq v11 pipelines to create stable, high-quality hydrology reconstructions across Kansas watersheds.

### Hypothesis  
A **weighted hybrid ensemble** using USGS/USACE observations and climate-driven inference (ERA5 + Mesonet) will outperform any single-source reconstruction on:

- RMSE  
- Seasonal stability  
- Extreme-event reconstruction  
- Basin-to-basin generalizability  

---

# üìö 2. Background

Hydrology reconstructions are necessary because:

- Many USGS gauges have discontinuities  
- USACE reservoir inflow/outflow data has gaps  
- Climate-era datasets help backfill early-century missing data  
- Hydrology supports:
  - Reservoir management  
  - Story Node environmental context  
  - Flood/drought analysis  
  - Climate‚Äìhydrology coupling in KFM pipelines  

HYDRO-EXP-002 tests the fusion model before hydrology pipeline deployment.

# üõ†Ô∏è 4. Methods

## 4.1 Preprocessing
- Standardize CRS (EPSG:4326)  
- Resample all flows to daily  
- Normalize units (cfs, inflow/outflow AF/d via conversion)  
- Identify missing periods ‚Üí mask  
- Remove storms >99th percentile (separate validation)  

## 4.2 Compute Partial Reconstructions
1. **USGS direct**  
2. **USACE storage inversion ‚Üí inflow/outflow**  
3. **ERA5 climate-forced prediction**  
4. **Mesonet precipitation-driven estimates**  

Each produces a candidate series.

## 4.3 Weighted Fusion
```
flow_fused = Œ£ (w_i * series_i)
```
Weights optimized via:

- Basin-wise RMSE  
- Seasonal cycle preservation  
- Cross-basin generalizability  
- Gradient-free Powell tuning  

## 4.4 Extreme Event Correction
Use residual correction model to restore:

- Peak flow accuracy  
- Flood signature sharpness  
- Drought persistence pattern  

## 4.5 Validation Framework
Metrics:

- RMSE  
- MAE  
- NSE (Nash‚ÄìSutcliffe)  
- Spatial error transfer across basins  
- Extreme sensitivity
