# ü§ñ **AI-EXP-003 ‚Äî Story Node v3 Generation Trial (v11 LTS)**  
`mcp/experiments/2025-11-05_AI-EXP-003.md`

**Purpose:**  
Evaluate the **AI-assisted Story Node v3 generator** used in Focus Mode v3 and narrative pipelines, focusing on factual grounding, provenance completeness, CARE compliance, and masking of sensitive heritage information.

# üìò 1. Objective / Hypothesis

### Objective  
Test the Focus Mode Transformer v3 + CrewAI narrative worker to generate **draft Story Node v3** outputs with:

- full CIDOC-CRM alignment  
- accurate spatial/temporal context  
- correct provenance references  
- CARE-aligned narrative tone  
- zero speculative statements  

### Hypothesis  
With strict context windows (2‚Äì3 hop Neo4j neighborhood), AI narrative generation will meet baseline accuracy and compliance thresholds and reduce human editing time.

---

# üìö 2. Background

Story Node v3 requires:

- OWL-Time temporal precision  
- GeoJSON/H3 spatial grounding  
- Provenance (PROV-O + OpenLineage)  
- Careful narrative style (non-speculative, neutral, culturally ethical)  
- Links to datasets, events, documents, and people  

Prior versions (v10.x):

- Produced factual drift  
- Used inconsistent lineage tags  
- Required heavy human editing  
- Risked narrative overreach for cultural material  

AI-EXP-003 is the first v11 test of the **governed narrative stack**.

- Fetch STAC/DCAT metadata


- Feed context + constraints into FMT-v3  
- Generate:
  - Context panel  
  - Timeline panel  
  - Map panel  
- Apply narrative safety filter:
  - block speculative claims  
  - enforce neutral tone  
  - disallow genealogical inference  
  - prevent colonial language biases
