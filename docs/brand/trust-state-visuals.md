# Trust-State Visuals

Document how trust signals are represented across the shell:
- `source_role`
- `rights`
- `sensitivity`
- `review_state`
- `freshness`
- `release_state`
- `correction_state`
- finite outcome (`ANSWER` / `ABSTAIN` / `DENY` / `ERROR`)

Conventions must include non-color cues and plain labels.
