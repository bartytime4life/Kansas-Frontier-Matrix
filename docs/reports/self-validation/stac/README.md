## ğŸ§¾ Report Details

### 1. ğŸ§  `_summary.json`
Contains combined results from both `pystac` and `stac-validator`.

| Field | Description | Example |
|--------|-------------|----------|
| `pystac.files` | Total files validated by PySTAC. | `52` |
| `pystac.failed` | Files failing PySTAC validation. | `0` |

```json

}
```

## ğŸ§© Validation Workflow Integration

**Workflow:** `.github/workflows/stac-validate.yml`

### Process
1. Discover all JSON files under `data/stac/`.  
2. Validate with **PySTAC** for structure and link integrity.  
3. Validate with **STAC Validator** for schema correctness.  
4. Combine results into `_summary.json`.  
5. Upload artifacts and append governance log.  
6. Update `focus-telemetry.json` with success/failure rates.
