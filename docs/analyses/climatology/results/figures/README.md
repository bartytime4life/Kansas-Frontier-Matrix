## üìñ Overview

The **Figures** directory houses high-fidelity visualizations generated from climatology analyses: temperature anomaly maps, precipitation change graphs, trend animations, projection slices, etc.  
Each visualization is accompanied by a descriptive caption, file format, resolution, provenance link (to methods + dataset), and version metadata.  
This README acts as an entry point for navigating the visual assets, understanding their context, and tracing their origin.
