## üìò Overview

The **Ecology Reports Module** consolidates analytical results and visualization outputs from ecological modeling workflows, including:
- Species distribution and richness analyses  
- Habitat suitability and fragmentation mapping  
- Ecosystem service evaluation  
- FAIR+CARE and ISO sustainability audits  

All reports adhere to FAIR+CARE governance, ISO 50001/14064 sustainability protocols, and WCAG 2.1 AA accessibility guidelines for inclusive data visualization.
