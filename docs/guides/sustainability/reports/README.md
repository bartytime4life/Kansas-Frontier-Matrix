## ‚öôÔ∏è Report Schema (Unified)

| Field | Description | Example |
|--------|-------------|----------|
| `report_id` | Unique identifier for sustainability audit | `"sustainability-report-2025-11-09-0003"` |
| `component` | System or pipeline audited | `"ETL Hydrology Pipeline"` |
| `standard` | ISO or FAIR+CARE framework applied | `["ISO 50001", "ISO 14064", "FAIR+CARE"]` |
| `metrics` | Energy, carbon, and sustainability results | `{ "energy_joules": 11.8, "carbon_gCO2e": 0.0052, "renewable_percent": 83 }` |
| `faircare_status` | Outcome of governance audit | `"Pass"` |
| `auditor` | FAIR+CARE Council or Sustainability Committee | `"FAIR+CARE Council"` |
| `timestamp` | UTC timestamp of audit completion | `"2025-11-09T12:00:00Z"` |
