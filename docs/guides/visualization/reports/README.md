## üìò Overview

This directory stores all **automated validation outputs** and **governance audit records** for visualization subsystems within KFM:
- MapLibre interface accessibility reports  
- Timeline and story playback audits  
- Explainability dashboard FAIR+CARE evaluations  
- UI telemetry and performance accessibility metrics  

All reports are generated by CI/CD validation workflows and synchronized with the **Governance Ledger** for ethical and technical transparency.

| `report_id` | Unique identifier for validation event | `"ui-audit-2025-11-09-001"` |

| `audit_type` | Type of validation performed | `"Accessibility"`, `"FAIR+CARE"`, `"Performance"` |
