## üìò Overview

This `archive/` directory preserves all **historical STAC validation logs** generated by automated pipelines and governance reviews from previous Kansas Frontier Matrix releases.  
Each archive file is digitally signed, timestamped, and verified using SHA-256 hashes stored within the Governance Ledger and Telemetry Dashboard.

Archives are immutable ‚Äî once published, they **cannot be altered or deleted**, ensuring long-term audit integrity and scientific reproducibility.

## ‚öôÔ∏è Automated Archival Workflow

Archives are automatically generated by the following CI/CD steps:

**Workflow:** `.github/workflows/stac-validate.yml`

**Archival Steps:**
1. Execute full STAC validation for all datasets in `data/stac/**`.
2. Package JSON results and NDJSON logs into `stac_validation_vX.Y.Z.tar.gz`.
3. Compute SHA-256 checksums for all included files.
4. Update governance and release manifest ledgers.
5. Upload artifact to GitHub Releases and documentation tree.
