# Publication

Minimal governed publication checklist for the hydrology proof slice.

1. Confirm schema wave files exist and are Draft 2020-12.
2. Validate JSON artifacts and fixtures.
3. Confirm release manifest carries proof quartet (`spec_hash`, `run_receipt_ref`, optional `ai_receipt_ref` when model-mediated, `attestation_refs`).
4. Publish only from governed lineage (`RAW -> WORK/QUARANTINE -> PROCESSED -> CATALOG -> PUBLISHED`).
