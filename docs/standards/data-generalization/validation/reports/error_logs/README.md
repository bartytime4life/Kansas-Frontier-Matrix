## üìò Overview

This directory stores **stderr and stdout diagnostic logs** generated by CI/CD workflows responsible for validating generalization tasks applied to sensitive datasets.

These logs capture:
- Spatial generalization logic failures  
- Temporal masking errors  
- CARE metadata or sovereignty compliance violations  
- Schema mismatches  
- Asset loading or parsing errors  
- Incomplete or malformed generalization bundles  

Error logs are retained for **audit reproducibility**, **Council review**, and **release diagnostics**.

All results feed into:
```
releases/v10.2.0/focus-telemetry.json
reports/audit/governance-ledger.json
```
