## Layer 20: Transparency Log + Public Verification Portal

This layer builds a **local** append-only transparency log and static verifier portal from Layer 19 disclosure packets. It does **not** publish remotely, provide legal certification, or mutate source disclosure packets.

### CLI examples
- plan-only
- append-packets
- build-portal
- full
- verify-log
- local-serve

See `soilgrids_transparency_portal.py --help` and `transparency/transparency_log_spec_example.json`.
