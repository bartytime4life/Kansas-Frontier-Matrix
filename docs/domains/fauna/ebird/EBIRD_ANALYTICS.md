# eBird Layer 13 Analytics and Insights

Layer 13 builds **descriptive public aggregate indicators** and **public-safe insight reports** from already-public Layer 6/8/12 artifacts.

- No eBird downloads, credentials, or API keys.
- No exact coordinates in public artifacts.
- No occupancy/abundance/true absence/population-trend/causal claims.

## CLIs
- `kfm-ebird-analytics`
- `kfm-ebird-insights`

## Safety
Public outputs keep `public_safe=true` and `exact_points=restricted`, enforce suppression minimums, and include interpretation warnings that changes are descriptive reporting only.
