# Core Entities

Starting baseline entities and minimum invariants.

## Entity set

- Place
- Event
- Observation
- Artifact
- Dataset
- DatasetVersion
- Agent
- Claim
- EvidenceBundle
- Relationship

## Invariant checklist

For each entity definition, document:

- Identity rule
- Time semantics
- Policy label propagation
- Evidence requirement
- Failure behavior (default-deny/fail-closed)
