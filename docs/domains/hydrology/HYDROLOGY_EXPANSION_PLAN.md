# Hydrology Expansion Plan

## Goal
Expand from fixture-first proof lane to broader governed hydrology coverage without weakening evidence guarantees.

## Phases
1. **Stabilize P0**: contracts, validators, fixtures, proof closure, release dry-run.
2. **Operationalize P1**: CI enforcement, correction drills, rollback drills, source onboarding playbook.
3. **Expand P2**: additional hydrology-adjacent lanes with explicit source-role separation.

## Exit criteria per phase
- Each phase requires passing offline tests, documented policy posture, and reproducible artifacts.
