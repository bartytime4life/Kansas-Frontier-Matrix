# Hydrology Continuity Inventory

Track continuity-critical artifacts for correction and rollback drills.

| Artifact class | Continuity requirement |
| --- | --- |
| Source descriptors | Versioned and review-auditable |
| Fixtures | Pinned and reproducible |
| Processed outputs | Deterministic and content-addressed |
| Run receipts | Immutable run history |
| Evidence bundles | Stable references from public claims |
| Decision envelopes | Finite outcomes and reason codes |
| Release manifests | Alias lineage and supersession history |
