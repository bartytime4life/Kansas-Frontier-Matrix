## üìö Overview

The **Legacy Archive** contains **all pre-MCP and early developmental documents** from the Kansas Frontier Matrix project (2017‚Äì2023).  
These records include concept drafts, early GIS integrations, and exploratory data workflows written before the project‚Äôs unification under the **Master Coder Protocol (MCP)**.

* üß† **Historical traceability** ‚Äî capturing the origins of MCP concepts.
