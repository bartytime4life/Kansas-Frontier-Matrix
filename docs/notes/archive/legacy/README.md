## üìö Overview

The **Legacy Archive** contains **pre-MCP and early developmental documents** from KFM (2017‚Äì2023).  
These records include concept drafts, early GIS integrations, ingestion prototypes, and field sketches written before unification under **Master Coder Protocol (MCP)**.

While not conforming to modern schemas or FAIR principles at creation, these files are preserved for:

* üß† **Historical traceability** ‚Äî capturing the origins of MCP concepts.  
* üßæ **Provenance linking** ‚Äî referencing early datasets, ETL prototypes, and GIS experiments.  
* üîó **Continuity** ‚Äî enabling modern documentation to reference design lineage.  
* üß± **Education** ‚Äî teaching how MCP and reproducible governance evolved.
