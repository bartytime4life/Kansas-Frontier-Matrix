`data/derivatives/terrain/`.  
Checksums ensure the **integrity, reproducibility, and provenance** of DEMs, hillshades, contour maps, and slope rasters used throughout the KFM ETL â†’ STAC â†’ Knowledge Graph â†’ Web UI pipeline.

ðŸ› ï¸ Generate & Verify

Generate (manual)

cd data/derivatives/terrain
for f in *.tif *.geojson; do
  [ -f "$f" ] && shasum -a 256 "$f" > "checksums/${f}.sha256"
done

Verify (portable)

cd data/derivatives/terrain
for c in checksums/*.sha256; do
  base=$(basename "$c" .sha256)
  sha256sum -c "checksums/${base}.sha256" || echo "Mismatch: $base"
done

Makefile Integration
