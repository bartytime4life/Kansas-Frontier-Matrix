# PMTiles delta slice release notes

This is a toy PMTiles v3 delta slice for validating the governed ingestion and attestation lane. It contains eight addressed tiles at z=8–9, a TileJSON manifest, and minimal vector-tile payloads under the `huc12` layer.

Publication posture: public-safe fixture only. It is not authoritative hydrologic data and must be replaced with governed source-derived output before production publication.
